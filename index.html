第十屆學生專案


BagKeeper
隨時隨地在任何一家店寄放你的個人物品，安全又自在。無論你將要旅遊、運動、約會....，寄物便不再讓多餘的包包雜物成為你生活中的累贅。

TechCare
Techcare長照守護家的使命是要讓所有需要被照顧的人得到妥善的照顧。我們透過科技建立長期照護的“信任”，同時使照顧者能更有效率的工作、使被照顧者能受到更好的照顧，並讓家屬能夠隨時隨地掌握被照顧者的情形，彷彿家屬和照顧者就在被照顧者身邊一起照顧一般。Techcare長照守護家以被照護者為中心，找回長照負擔中應有的溫馨。

AdCharity
台灣的愛心捐款雖不間斷，但社福界的募款總和卻有限，造成部分社福團體與公益專案找不到贊助，使得弱勢者能得到的捐助大幅減少。 我們團隊想幫助社福團體籌募更充裕的資金，進而順利地協助弱勢者。 為了解決捐款的排擠效應，我們規劃出「看廣告做公益」的方式，將大家收看我們廣告的收益，捐給你們想幫助的團體，達到永續照顧社會的目的。

Credit4U-代刷幫手
是整合消費者及信用卡持卡方的代刷媒合服務平台。協助消費者在購物時能靈活地任意選擇對消費項目有利的各種信用卡優惠，並幫助信用卡持卡方取得刷卡額度的回饋機制。解決信用卡長久以來使用上的封閉消費模式，能有效地使消費者與持卡人雙方皆能以最少的資源透過共享經濟的加乘效果而相互受惠。

Outfit Hunt
Outfit hunt是一個讓妳用現有衣服找穿搭的實用APP，如果妳有一件／很多件不知該怎麼搭配的衣服，outfit hunt透過精準的服裝掃描分析技術搭配最流行、最潮的穿搭達人組合，將會幫妳解決不知該搭什麼的煩腦，用現有的衣服就能讓穿搭天天都不無聊。 if you have a particular item and don’t know how to style it you can start outfit hunter."
